## Overview
This lab simulates a basic corporate Active Directory environment by deploying a Windows Server domain controller and joining a client system to the domain.

## What I Did
- Deployed a Windows Server virtual machine as a domain controller
- Installed and configured Active Directory Domain Services (AD DS)
- Created and managed domain users
- Configured basic networking to allow domain and internet access
- Joined a Windows client system to the domain
- Verified domain connectivity and user authentication

## What This Demonstrates
- Understanding of Active Directory and domain-based networking concepts
- Familiarity with domain controllers and client systems
- Ability to manage users in a centralized directory service
- Awareness of authentication and access control fundamentals
- Experience working within a virtualized lab environment
