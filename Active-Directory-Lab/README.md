# Active Directory Lab

This project demonstrates the setup and administration of an on-premises Active Directory environment using Windows Server.

The lab simulates a small enterprise network with domain services, user management, and authentication controls.

## Technologies Used
- Windows Server (Active Directory Domain Services)
- Windows 10/11 Client VM
- Azure Virtual Machines
- PowerShell

## Key Skills Demonstrated
- Active Directory configuration and management
- User and group administration
- Domain-joined systems
- Authentication and access control concepts
- Basic PowerShell usage

## Lab Overview
- Deployed a Windows Server domain controller
- Configured Active Directory Domain Services (AD DS)
- Created and managed users and groups
- Joined a client machine to the domain
- Validated authentication and permissions
